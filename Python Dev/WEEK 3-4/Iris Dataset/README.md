# Iris Flower Classifier (Decision Tree)

## Overview

This project uses a decision tree classifier from the `scikit-learn` library to classify iris flowers into one of three species: setosa, versicolor, or virginica. Itâ€™s built on the classic Iris dataset and is designed to give a simple, clear introduction to machine learning using Python.

## Key Features

- Loads and splits the Iris dataset into training and testing sets
- Trains a DecisionTreeClassifier using scikit-learn
- Predicts species of unseen iris flowers based on petal/sepal size
- Outputs prediction accuracy and sample predictions
